Propriétés publish (dans le PropertyGroup conditionné `Condition="'$(PublishSingleFile)' == 'true'"`) :
- SelfContained = true          (explicite : RuntimeIdentifier n'implique plus self-contained depuis .NET 8)
- RuntimeIdentifier = win-x64
- PublishSingleFile = true
- IncludeNativeLibrariesForSelfExtract = true   (WPF embarque des DLL natives → vrai mono-fichier)
- EnableCompressionInSingleFile = true          (~140 Mo → ~60-70 Mo)
- PublishTrimmed = false                         (WPF n'est PAS trim-safe — crash au lancement sinon)
- PublishReadyToRun = true                       (démarrage autostart plus rapide) — À AJOUTER
- InvariantGlobalization = false                 (formatage fr-FR des comptes à rebours) — À AJOUTER

Commande de publication canonique (verbatim STACK.md / CONTEXT) :
  dotnet publish src/Chronos/Chronos.csproj -c Release -r win-x64 -p:PublishSingleFile=true --self-contained true

Sortie attendue :
  src/Chronos/bin/Release/net8.0-windows/win-x64/publish/Chronos.exe   (un seul exe, ± .pdb)
</config_verrouillée>

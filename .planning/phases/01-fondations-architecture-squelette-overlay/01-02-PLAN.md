<objective>
Rendre le Topmost de l'overlay durable : `Topmost=True` n'est pas garanti dans le temps (plein écran
tiers, réagencement du Z-order, sessions longues). Ce plan ajoute la réaffirmation périodique de la
bande topmost via `SetWindowPos(HWND_TOPMOST, SWP_NOACTIVATE)` — sans jamais voler le focus (ROB-04).

Purpose : sans réaffirmation, l'overlay retombe derrière les autres fenêtres et cesse d'être un
instrument « toujours visible ». Output : `TopmostGuard` piloté par un DispatcherTimer dédié (~2 s),
attaché à la fenêtre au bon moment (HWND présent), testable via un délégué P/Invoke injectable.
</objective>

    ```csharp
    using System.Runtime.InteropServices;

    namespace Chronos.Interop;

    internal static class NativeMethods
    {
        // hWndInsertAfter : -1 = HWND_TOPMOST (place/maintient dans la bande topmost)
        public static readonly IntPtr HWND_TOPMOST = new(-1);

        public const uint SWP_NOSIZE     = 0x0001;  // ne pas redimensionner
        public const uint SWP_NOMOVE     = 0x0002;  // ne pas déplacer
        public const uint SWP_NOACTIVATE = 0x0010;  // NE PAS activer → aucun vol de focus

        [DllImport("user32.dll", SetLastError = true)]
        [return: MarshalAs(UnmanagedType.Bool)]
        public static extern bool SetWindowPos(
            IntPtr hWnd, IntPtr hWndInsertAfter,
            int X, int Y, int cx, int cy, uint uFlags);
    }
    ```

    2. `src/Chronos/Services/TopmostGuard.cs` — VERBATIM RESEARCH.md § « Services/TopmostGuard.cs ». Points obligatoires :
    - `IDisposable`, DispatcherTimer dédié `Interval = TimeSpan.FromSeconds(2)` (distinct du futur tick d'interpolation Phase 4-5) ;
    - délégué injectable `public delegate bool SetWindowPosFn(...)` avec défaut `= NativeMethods.SetWindowPos` (rend les flags vérifiables en test) ;
    - `Attach(Window window)` : `_hwnd = new WindowInteropHelper(window).EnsureHandle();` puis `Reassert();` puis `_timer.Start();` ;

    - `Dispose() => _timer.Stop();`.
    NE PAS utiliser le toggle `Topmost=false;Topmost=true` (scintillement + réactivation). `SetWindowPos` + `SWP_NOACTIVATE` uniquement.
